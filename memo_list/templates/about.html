About Memo App

Memo is an application that you can use to keep track of notes / shopping lists or tasks more easily.

You can add or delete items, mark them as done or undone and you can see the date you posted the items.

Python, Django, Bootstrap and CSS were used to develop the application and all data is saved in the Sqlite3 database.
